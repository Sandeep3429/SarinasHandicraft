Steps for installing and running application.
Pre-require software
i.  Download and install Visual Studio 2019
ii. Download and install MS-SQL server
iii.Download and install MS-SQL server managment studio

1. Download the zip file.
2. Extract the zip file.
3. Run Visual Studio 2019.
4. Import the unzipped project on Visual Studio 2019
5. Run the application.
6. The application should open on browser
